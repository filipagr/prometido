## Notas gerais

- Programas com `.txt` em vez de `.pdf`: conteúdo equivalente, extraído de páginas web arquivadas.
- Para o cowork com `/extract-pdf`, ficheiros `.txt` são passados directamente ao Claude sem pdftotext.
- Links do Wayback Machine: usar o formato `https://web.archive.org/web/{timestamp}if_/{url}` para download directo de binários.
- O Arquivo.pt (arquivo.pt) tem cobertura mais completa para domínios portugueses — preferir sobre o Wayback Machine quando disponível.
